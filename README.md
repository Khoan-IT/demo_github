# Github Tutorial

## Yêu cầu

- Python >= 3.9

## Cài đặt

```bash
git clone <repo-link>
cd <repo-folder>
````

Cài đặt các thư viện cần thiết:

```bash
pip install -r requirements.txt
```

## Cách chạy

Chạy chương trình chính bằng lệnh:

```bash
python main.py
```

## Cấu trúc thư mục

```bash
.
├── main.py
├── requirements.txt
└── README.md
```
